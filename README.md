# Demo Styleguide
> Demo styleguide

Demo styleguide implementation using [Marvel App Styleguide](https://marvelapp.com/styleguide/overview/introduction)

![](https://image.ibb.co/iLqX1V/screencapture-localhost-9001-2018-10-28-18-38-54.png)

## Installation

OS X & Linux:

```sh
npm install
```


## Development setup

Run storybook.

```sh
npm run storybook
```
